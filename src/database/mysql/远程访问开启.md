### 开启mysql的远程访问 两种方式
* 方式1
```shell
grant all privileges on *.* to 'root'@'192.168.1.145' identified by '123456' with grant option;
```
```shell
flush privileges;
```
* 方式2
```shell
use mysql;
update user set host = '%' where user = 'root';
```
 **更细权限之后，一定要刷新一下，否则不会生效**
```shell
flush privileges;
```

* 方式3
```shell
  此处为设置root用户远程
  mysql -uroot -p  #输入密码

  use mysql;

  update user set host = '%' where user = 'root';

  select host, user from user;

  flush privileges;
```
