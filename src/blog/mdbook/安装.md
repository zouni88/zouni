### 安装
* 要先安装cargo,通过`cargo`命令安装`mdbook`
```shell
# 安装
cargo install mdbook
# 卸载
cargo uninstall mdbook
# 安装最新版本
cargo install --git https://github.com/rust-lang/mdBook.git mdbook
```
* 同样，确保将 Cargo bin 目录添加到您的 .PATH 环境变量中

### create book
创建一本书先

    mdbook init my-first-book

    cd my-first-book

    # 启动服务，默认打开浏览器
    mdbook serve --open

### 发布

    mdbook build
