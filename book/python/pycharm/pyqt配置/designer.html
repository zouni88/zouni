  2. 找到desiginer.exe位置,我的是在anaconda3安装的，就在我的虚拟环境目录下面找到：

Working directory 这个路径是工作路径，所以一般就填入$FileDir$

  1. 编辑完UI后，还需要转换成py文件才行，所以需要配置pyui转换

需要配置下参数：

Program: python.exe 路径

Working directory: $FileDir$
